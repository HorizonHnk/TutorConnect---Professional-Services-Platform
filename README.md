# TutorConnect - Professional Services Platform

> A comprehensive tutoring and professional services marketplace connecting students with expert tutors and service providers.

### 📺 Video Tutorial Series
[Watch the Complete Build Series](https://www.youtube.com/playlist?list=PLrZbkNpNVSwwbf-xK9LTqoFwbl-CdnG_Y)

### 🚀 Overview
A full-stack platform revolutionizing the way students connect with tutors and technical service providers. Built with modern technologies and designed for scalability.

### ⭐ Key Features
- Real-time session scheduling and management
- Multi-category service marketplace
- Secure payment processing
- Course catalog and management
- Service provider profiles and ratings
- Student dashboard
- Advanced booking system

### 💻 Tech Stack
- Frontend: ReactJS + Vite
- Backend: Node.js
- Desktop: Electron
- Mobile: Android Studio & Xcode
- Cloud: Firebase + Google Cloud AI

### 📦 Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/tutorconnect.git

# Navigate to project directory
cd tutorconnect

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Run development server
npm run dev
```

### 🛠️ Environment Setup
Create a `.env` file with the following variables:
```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```
